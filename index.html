Instructions:

[Arrow Keys] - Move the player (or drag them with mouse)
[Mouse] - Drag "anything" around
[Space] - Toggle Bullet Time (edit while paused)
[0-9] - Spawn objects
[A] - Spawn more Appels
[X] - Delete an object
[M] - Ragdoll (lag!!)
[F] - Fix a block in place / or constrain a pin
[P] - Add / Remove a pin or press while dragging to pin objects together
[N] - Next scene
[R] - Reset scene
[G] - Cycle gravity modes
[S] - Show sleeping objects
[O] - Output/Save
[I] - Input/Load
